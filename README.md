# Webpack-Babel-Typescript-Jest

> Template for Webpack/Babel for typescript project and use of Jest for testing.

## Run Development Server

```shell
yarn dev
```

## Building the repo

```shell
yarn build
```

## Building bundles

```shell
yarn build:webpack
```

## Building js files

```shell
yarn build:js
```

## Building types only

```shell
yarn build:types
```

## Type-Checking the repo

```shell
yarn type-check
```

## Testing

```shell
yarn test
```

## Clear all build files (Deletes)

```shell
yarn clean
```
